3. **Enable** in WoW Classic AddOns menu
4. **Configure** with `/wdts` command

## ‚ú® Key Features

- ‚úÖ **100% Offline** - All translation happens locally in-game
- ‚úÖ **Dependency Parsing** - Structure-based translation for accurate word order (v0.5.0)
- ‚úÖ **Auto-Translation** - Automatically translates when confidence is sufficient
- ‚úÖ **Intent Detection** - Understands LFG, invites, ready status, etc.
- ‚úÖ **Safe & Conservative** - Stays silent when uncertain
- ‚úÖ **Configurable** - Choose which channels to translate

## üíª Basic Usage

**Commands:**
- `/wdts` - Open configuration window
- `/wdts test <message>` - Test translation
- `/wdts debug` - Show translation history

Translations appear automatically in chat when confidence ‚â• 0.50 (green for high confidence, yellow for medium).
